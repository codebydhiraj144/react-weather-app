# react-weather-app
A React-based weather application to search current weather information for any city using the OpenWeather API.  
Displays temperature, feels-like temperature, humidity, and weather condition, with dynamic images based on weather.



## Features

- Search weather by city name
- Displays temperature, feels like temperature, humidity, and condition
- Dynamic background image changes based on weather (hot, cold, rainy, cloudy, clear)
- Error handling for invalid city names
- Responsive UI using React and CSS
